
============================
EXEMPLE : CLASSES ET ROUTAGE
============================

Exemple de projet simple qui illustre le fonctionnement des objets (classes) via un système routé.

Pour comprendre le routage sur PHP voir l'article ci-dessous
(ici avec le framework LARAVEL) :

https://apcpedagogie.com/routage-sous-laravel/


Pour faire fonctionner ce script il est INDISPENSABLE de modifier 
dans le fichier .HTACCESS, la ligne : 

#Répertoire racine par défaut :
RewriteBase /08-route/

Ici on considére le repertoire par défaut du site à cet emplacement :
/08-route/

Ceci est à modifier en fonction de son utilisation.

Par exemple a la racine du serveur :
/

Ou dans un dossier : test
/test/

Dans tous les cas, la supression du fichier ".htaccess" va générer un nouveau fichier
correctment routé.
